
# Student Finance Tracker - Transformation Plan

## Phase 1: Database Schema
Create tables for:
- `transactions` (income/expenses with categories, dates, notes)
- `budgets` (monthly category budgets)
- `savings_goals` (name, target, deadline, current amount)
- `badges` (achievement tracking)

## Phase 2: Update Onboarding
- Student-focused fields (allowance, expense categories, savings goals)
- Simple, beginner-friendly language
- Remove complex financial terms

## Phase 3: New Dashboard
- Total balance, monthly spending summary, savings progress
- Budget usage bars with visual analytics
- Charts (pie for expenses, bar/line for trends)

## Phase 4: Core Features
- Add Income/Expense forms with category, date, notes
- Budget management with progress bars and alerts
- Savings goals with visual progress
- Search and filter for transactions

## Phase 5: Analytics & Guidance
- Spending analytics with charts
- Student-friendly financial tips
- Highlight highest spending category

## Phase 6: Gamification & Profile
- Badge/achievement system
- Profile page with stats and achievements

## Phase 7: Export & UI
- CSV/PDF export
- Dark mode toggle
- Modern student-friendly UI
- Remove FIRE planner, tax planner references

## Phase 8: Cleanup
- Remove/simplify complex financial pages
- Update navigation and routing
